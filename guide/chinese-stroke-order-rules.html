Chinese Stroke Order Rules Explained

Chinese characters are written in a fixed order of strokes. Learn the simple rules — with examples — and you can work out the correct order for almost any character.

Every Chinese character is written in a set order of strokes. It can feel arbitrary at first, but stroke order isn't decoration — it makes characters faster to write, better balanced, and much easier to remember. Once you learn a handful of rules, you can work out the order for most characters you meet.

Why stroke order matters

  • Speed and flow. The correct order lets your hand move naturally from one stroke to the next.
  • Balance. Characters come out evenly proportioned instead of lopsided.
  • Memory. Writing in a consistent order turns a character into a single motion you can recall, not a picture you have to copy.
  • Legibility. Fast handwriting and cursive only look right when the underlying stroke order is correct.

The main rules

1. Top to bottom

Write upper strokes before lower ones. In 三 (sān, three), you write the top line, then the middle, then the bottom.

2. Left to right

Write left strokes before right ones. In 人 (rén, person), the left-falling stroke comes before the right-falling stroke.

3. Horizontal before vertical

When strokes cross, write the horizontal first. In 十 (shí, ten), the horizontal line comes before the vertical.

4. Left-falling before right-falling

Diagonal strokes that fall to the left are written before those that fall to the right, as in 八 (bā, eight).

5. Outside before inside

Draw the outer frame before what goes inside it. In 月 (yuè, moon), the outer shape is written before the two inner lines.

6. Inside, then close the box

When a character is fully enclosed, write the sides, then the contents, then seal the bottom last. In 国 (guó, country), you draw the enclosure, then the inside (玉), then the closing bottom stroke.

7. Centre before wings

In symmetrical characters, the central stroke often comes first. In 小 (xiǎo, small), the middle stroke is written before the two side dots.

8. Dots and minor strokes last

Small finishing strokes, like the dot in 玉 (yù, jade), are usually added at the end.

These rules cover the large majority of characters. Where a character seems to break them, it's usually following a more specific version of the same idea — so when in doubt, apply "top to bottom, left to right" and you'll rarely be far wrong.

Common mistakes to avoid

  • Closing a box too early. Always fill the inside before sealing the bottom.
  • Starting from the right. Our instinct varies, but Chinese flows left to right and top to bottom.
  • Guessing on every character. Learn the rules once and they transfer everywhere.

Practise it

The best way to lock in stroke order is to write slowly and deliberately. Generate a practice sheet in the Worksheet Maker, trace each character while saying the strokes in order, then write it from memory. Start with simple, high-frequency characters like those in the HSK 1 set and the common radicals.
