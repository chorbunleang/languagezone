How Chinese numbers combine

Once you know 1–10, the rest follow a simple pattern:

  • 11–19 = 十 (ten) + the digit. So 11 is 十一 (shí yī), 15 is 十五 (shí wǔ).
  • Tens = the digit + 十. So 20 is 二十 (èr shí), 50 is 五十 (wǔ shí).
  • Two-digit numbers combine both. 42 is 四十二 (sì shí èr) — literally “four ten two.”
  • 100 is 一百 (yī bǎi), “one hundred.”
Because every number is built from these eleven characters, this is one of the highest-value worksheets a beginner can practise. Learn to write them cleanly and you can write any number with confidence.
